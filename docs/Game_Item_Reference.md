# Game Item Reference

A complete list of all items, organized by rarity tier, with their trigger condition and effect (or stat bonus).

---

## Legendary

| Item Name | Condition → Effect |
| --- | --- |
| **Blade o' Wrath** | Kill Enemy → Gain a stack of +1 Flat Damage (max 7 stacks); stacks reset on Rest |
| **Crown o' Pride** | Not hit this turn → Heal 10% HP |
| **Boots o' Lust** | Trigger a Status effect → Status triggers twice |
| **Ring o' Greed** | Gain Coins → Coins gained +20% |
| **Gauntlet o' Sloth** | Per turn without moving → +1 Lazy Stack (max 5). Next attack: consume all stacks for +[stack]d4 |
| **Clock o' Chronos** | Die → Revive with 1% HP (1x per Combat) |
| **Mask o' Gluttony** | Kill Enemy → +10 HP |
| **Robe o' Envy** | Hit Enemy → +5 to a Random Stat (lasts 1 turn) |
| **Grid o' Topos** | Hit Enemy → Swap positions with the enemy |

### Notes
- "Bisa Muncul Berkali Kali" = "Can appear multiple times" — these items can be drawn/obtained more than once, unlike the standard unique items in each tier.
- Stat abbreviations: STR (Strength), LCK (Luck), ACC (Accuracy), INT (Intelligence), MOV (Movement), VIT (Vitality), ATT (Attack), DEX (Dexterity), CON (Constitution), AC (Armor Class), HP (Health Points).
